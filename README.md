# Movie Finder

Movie Finder is a React app for browsing movies and TV shows, searching titles, viewing details, and saving favorites locally.

## Features

- Browse trending and top-rated movies/TV shows
- Search movies and shows
- View item details and related content
- User authentication flows (login/signup)
- Favorites list stored in local storage

## Tech Stack

- React + React Router
- Create React App tooling
- Axios for API requests
- Firebase SDK

## Getting Started

### 1) Install dependencies

```bash
npm install
```

### 2) Start development server

```bash
npm start
```

App runs at `http://localhost:3000`.

## Scripts

- `npm start` — run the app in development mode
- `npm test` — run tests
- `npm run build` — create a production build

## Notes

- This project currently routes the app under `/react-movie-app`.
